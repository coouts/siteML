Quem Somos

No mercado de Tecnologia da Informação desde 2021, a TECRLA possui uma filosofia de trabalho consistente e fundamentada, com o objetivo de passar conhecimento para todos os jovens do Brasil. Ensinamos Desenvolvimento Web (incluindo a linguagem HTML e CSS) e Sistemas Operacionais, através de aulas online. Temos como principal missão o comprometimento com o ensinamento para garantir melhores resultados em seu conhecimento, para no futuro ter um bom retorno no seu sucesso!
Curiosidades: O nome do site foi pensado, com base nas iniciais das criadoras.
TEC = tecnologia;
R = Rafaela;
L = Leticia;
A = Última letra do nome das duas (leticiA, rafaelA).
