Como criar uma tabela em HTML

Quando se tem um conjunto de dados e precisa-se apresentá-los ordenadamente, uma boa maneira de fazê-lo é usando as tabelas. Elas são basicamente compostas de linhas (‘tr’..’/tr’) e colunas (‘td’...’/td’) cruzadas entre si. Mas os cabeçalhos da tabela devem ser apresentados não em “td”, mas em “th”; caso não, alguns códigos ficam comprometidos a não funcionarem. A tag básica das tabelas é a “table” “/table”, ou seja, toda a informação da tabela deve estar contida entre os delimitadores dessa tag.

Como inserir uma imagem em HTML

Para inserir uma imagem no HTML basta utilizar a tag “img” com o atributo src. Ou seja: o atributo src, ou source, vai conter a url da imagem que será inserida.
Diante disso, a sintaxe final será:
“img src=url"

Para inserir uma imagem local, podemos apenas incluir o nome da imagem com sua extensão, como no exemplo abaixo:
“img src=imagem.jpg"

Se ela estiver contida dentro de uma pasta local, nós devemos referenciar a pasta também. Por exemplo, se minha página principal está contida na pasta public_html e dentro dela temos uma pasta chamada imagens, onde está a nossa imagem, utilizaremos a seguinte url:
“img src=imagens/imagem.jpg"

Agora digamos que temos uma pasta public_html onde dentro dela temos as pastas imagens e pages, onde a nossa página esta dentro da pasta pages e a imagem está dentro da pasta imagens. Dessa forma, utilizamos ../ para poder voltar uma pasta e assim entrar na pasta que queremos.
“img src="../imagens/imagem.jpg"

Por outro lado, podemos ainda incluir a url completa da nossa imagem. Digamos que nosso domínio é https://meudominio.com e utilizando o exemplo anterior. Dessa forma, utilizaremos o seguinte código:
“img src=https://meudominio.com/public_html/imagens/imagem.jpg “.

Como criar uma lista Horizontal em CSS

Primeiramente montamos a estrutura básica do HTML e ligamos o arquivo HTML com o do CSS, usando a tag “link”. Feito isso, podemos começar a estruturar lista horizontal.
Dentro do body, começa montando uma div que será ID e na parte do CSS, vamos começar com a #, dentro dela vamos dar uma largura e altura, nossa caixa principal estará feita. Agora para colocarmos a lista dentro dessa div, vamos começar com a tag “ul” que irá representar uma lista desordenada e nela também vamos colocar como ID. Na parte do CSS, vamos colocar a # e o nome que foi dado, colocando apenas um tamanho. E para finalizar, criaremos nossas “li” que será os itens desta lista dentro da “ul”. Nesse caso, iremos colocar nossas “li” como class. Vamos colocar uns 5 itens nessa list, tudo com class e com o mesmo nome. Na parte do CSS, vamos começar colocar um ponto, que significa que ali foi usado o class, abre as chaves e logo em seguida colocamos uma largura, altura e daremos um float: left; que irá permitir com que os itens fique na horizontal.

Como colocar uma imagem de Background em CSS

Bom, primeiramente, não esquecendo da estrutura básica do html, logo em seguida, no body, iremos abrir uma div. Feito isso, no CSS, iremos começar com a #.
Em seguida irá fazer conforme está abaixo:

“background-image: url(../imagens/prog.jpg);”

Onde no caso o “(../imagens/” é o nome da pasta no qual você salvou a imagem e o “prog.jpg” é o nome da imagem e a extensão dela.
Se caso quiser deixar a imagem como fundo, basta colocar o comando background –size: cover; e pronto, tudo feito.
